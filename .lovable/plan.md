# 간격 조정 계획 (모바일 상단 메뉴)

모바일 화면(소유자 앱)에서 상단 메뉴(대시보드, 대여현황, 추이분석)가 좁은 화면에서도 가로 스크롤 없이 한눈에 더 여유롭게 확인될 수 있도록 다음과 같이 간격을 조정하겠습니다.

### 기술적 상세 내역

1. **`src/routes/_authenticated/my.tsx` 파일 내 `SegmentedTabs` 컴포넌트 개선**
   - 모바일 화면에서 탭 버튼의 좌우 패딩과 아이콘과의 간격을 최적화합니다.
     - 기존: `px-3.5 py-1.5 text-sm gap-1.5`
     - 변경: 모바일에서는 `px-2.5 py-1.5 text-xs gap-1`, 데스크톱(sm 이상)에서는 기존 크기(`sm:px-3.5 sm:py-1.5 sm:text-sm sm:gap-1.5`) 유지
   - 모바일에서 텍스트와 아이콘 크기를 살짝 조절하여 직관성을 높이면서 가로폭 공간을 확보합니다.

2. **상단 헤더 내부 레이아웃 여백 미세 조정**
   - 로고와 탭 메뉴, 로그아웃 버튼 사이의 간격(`gap-4`)을 모바일에서는 `gap-2`로 조정하여 더 넓은 메뉴 공간을 제공합니다.

이 방향으로 진행해도 괜찮으실까요? 확인해 주시면 바로 반영하겠습니다.
